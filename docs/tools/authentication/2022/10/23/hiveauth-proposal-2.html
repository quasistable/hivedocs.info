    Hive Authentication Services Development and Maintenance

    Oct 23, 2022 • arcange

    ![](https://i.imgur.com/QnlB5h6.png) The Hive Authentication Services have been officially launched several months ago and I would like to thank everyone who has given their support and enabled its development. ### What are Hive Authentication Services? ![](https://i.imgur.com/dLPYaIT.png) Hive Authentication Services (HiveAuth) makes it easy and secure for users to authenticate without providing any password or private key • on any device (mobile, tablet, desktop, …) • on any operating System (Android, iOS, Windows, macOS, Linux, …) • on any applications (either web, desktop or mobile) As of today, HiveAuth provides the most simple and secure 3FA authentication process. HiveAuth solves password problems and is phishing-proof because you no longer need to provide your password or private key to any HiveAuth-enabled app or website. ![](https://i.imgur.com/ZZhk0cv.png) More than 10K users have used it at least once to authenticate with their favorite application and the HiveAuth infrastructure handles more than 9K requests per day (authentications or transactions). And it's growing every day! ### Work done A lot has happened since the first announcement and if you have regularly followed updates regarding HiveAuth, you know that we have never stopped working on the project, both visibly by implementing new features and behind the scenes providing help for app developers. Here is a quick recap of the work that has been done: * [The very first Annoucement](/@arcange/hive-authentication-services-proposal) * Creation of a guide for developers Hive Authentication Services has become the first fully decentralized authentication service backed by a blockchain. It enables the concept of ***"Your Hive account is your key"*** to become a reality, both for the Hive ecosystem and for the "outside world". HiveAuth opens the door to countless possibilities. It also highlights the many functionality and ease of development on Hive. Recent presentations to audiences outside of the Hive ecosystem have clearly demonstrated that this is not only attracting audience interest in HiveAuth but also in the Hive blockchain itself. ### Budget For this new proposal, we are applying for a daily budget of 325 HBD for a period of 12 months. ### What's the funding for? There is still a lot of work to be done, in terms of development, support and communication. **1. Development roadmap** * support for multi-sig transactions (WIP) * support for variable timeouts for requests * support for signing transactions with multiple operations * support for signing transactions with multiple accounts * support for integration into existing new Apps and PKSA * support for multiple HAS nodes to allow load balancing and failover (WIP) * improve the redundancy and the scalability of the infrastructure (WIP) * dockerization of the HAS nodes for easy deployment * creation of a HAS node deployment tutorial * creation of new tutorials and code examples for additional development languages and much more... **2. Regular work** * Code maintenance and deployment * Support to users and (d)Apps developers * Documentation maintenance, both for users and developers * Communication and external awareness **3. Infrastructure** * Server(s) + Firewall * Security management and monitoring tools * Performances monitoring tools * Domain registration * Backup services ## Support I have always been easy to reach, responsive and as helpful as possible.
