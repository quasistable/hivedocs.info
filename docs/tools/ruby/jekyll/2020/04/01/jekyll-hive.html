    # [Jekyll::Hive](https://github.com/inertia186/jekyll-hive) Liquid tag for displaying Hive content in Jekyll sites: `{% hive %}`. ## Installation Add this line to your application's Gemfile: ```bash gem 'jekyll-hive' ``` And then execute: ```bash bundle ``` Or install it yourself as: ```bash gem install jekyll-hive ``` Then add the following to your site's `_config.yml`: ```yml plugins: - jekyll-hive ``` ðŸ’¡ If you are using a Jekyll version less than 3.5.0, use the `gems` key instead of `plugins`. ## Usage Use the tag as follows in your Jekyll pages, posts and collections: ```liquid {% hive author/permlink %} ``` This will place the associated content on the page. ## Jekyll Build When building your site with jekyll, you can continue to use the default command: ```bash jekyll build ``` If you would like to provide an alternate node: ```bash NODE_URL=https://anyx.io jekyll build ``` ## Contributing 1.
